# P28 toScreen

`toScreen` is a Screen Studio style product focused on screen recording plus AI-assisted software promo video editing.

> [!NOTE]
> This project is built on top of the [OpenScreen](https://github.com/siddharthvaddem/openscreen) base. 
> Base code has been migrated and is being extended with AI-driven automation.

## Positioning

- Record screen activity and turn it into polished product demo videos
- Use AI to reduce manual editing workload
- Emphasize automatic zoom, focus guidance, and presentation quality

## Core Workflow

1. Record the screen
2. Import or review the captured session
3. Apply AI auto-editing and smart zoom suggestions
4. Refine timeline, annotations, and framing
5. Export a software showcase or walkthrough video

## Core Features

- Screen recording
- Cursor/action-aware auto zoom (In Development)
- AI-assisted trimming and clip cleanup (Planned)
- Software demo composition
- Promo video export presets
